1. 프로젝트 개요 (Project Overview)
장르: 1인칭 2D 포인트 앤 클릭 방탈출 게임 (Point & Click Escape Room)

컨셉: 여러 방탈출 테마를 모아둔 '방탈출 카페' 플랫폼. 본 프로젝트는 첫 번째 테마인 스릴러 방탈출 구현에 집중함.

핵심 개발 방향: 드래그 앤 드롭 등 복잡한 물리/충돌 판정을 배제하고, 화면 클릭을 통한 UI 팝업(비밀번호 패드, 다이얼 등)과 텍스트/숫자 입력 위주의 두뇌형 퍼즐로 구성. 단일 씬(Single Scene)에서 조명만 바꾸며 진행.

2. 씬(Scene) 구성 및 UI/UX 명세
[Scene 1] 타이틀 로비 (Title_Scene)
비주얼: 힙한 느낌의 방탈출 카페 카운터 전경. 네온사인 로고와 대형 마스터키 소품.

BGM: 잔잔한 카페 재즈 음악.

UI 버튼 및 상호작용 (Hover/Click):

[테마 입장], [환경설정], [퇴근] 3개의 텍스트 버튼.

Hover 시: 텍스트가 노란색으로 변하며 크기 1.1배 확대. 가벼운 '틱' 효과음.

Click [테마 입장]: 종소리(Call Bell) 출력 ➔ 화면 전체가 1.5초간 검은색으로 페이드 아웃(Fade-out) ➔ ThemeSelect_Scene 로드.

[Scene 2] 테마 선택 (ThemeSelect_Scene)
비주얼: 어두운 복도 벽면에 핀 조명을 받는 3개의 테마 포스터 액자가 걸려 있음.

BGM: 긴장감 있는 앰비언트 사운드로 변경.

포스터 상호작용:

Case 01 (살인마의 암실): Hover 시 밝기 100%, 붉은색 외곽선 Glow, 1.05배 확대, 종이 바스락 소리. Click 시 화면 중앙에 [테마 정보 팝업] 활성화.

Case 02, 03 (잠김): 흑백 처리, [LOCKED] 이미지 오버레이. Hover 시 띡! 에러음 및 0.2초간 흔들림(Shake). Click 반응 없음.

[테마 정보 팝업] 패널:

팝업이 통통 튀며(EaseOutBack) 등장, 배경은 60% 어두워짐(Dimmed).

텍스트: 시놉시스와 난이도 표시.

Click [테마 시작]: 즉시 팝업 닫힘 ➔ 무거운 철문 닫히는 사운드 ➔ 화면 빠르게 암전(Fade-to-Black) ➔ MainGame_Scene 로드.

[Scene 3] 메인 게임: 살인마의 암실 (MainGame_Scene)
시각적 구성 (1인칭 고정 뷰):

좌측: 화학 약품 캐비닛(5자리 알파벳 자물쇠), 찢어진 주기율표 포스터.

중앙: 책상, 낡은 PC, 탁상 달력, 조명 스위치.

우측: 소형 금고(숫자 패드), 코르크 보드, 철문(풀 키보드 도어락), 사진을 너는 빨랫줄.

핵심 시스템 [조명 전환 기믹]:

방 중앙의 **'조명 스위치'**를 클릭할 때마다 방의 렌더링 상태가 [적색등 모드] ↔ [UV 블랙라이트 모드]로 토글(Toggle) 됨.

UV 모드일 때만 특정 오브젝트(벽면 마방진, 사진 뒷면 형광 글씨)가 활성화(Visible) 됨.

3. 퍼즐 플로우 및 정답 로직 (State Machine)
모든 퍼즐은 팝업 UI를 띄워 **'정답 문자열(String)'**을 입력받고 검증하는 방식으로 작동합니다.

🧩 퍼즐 1: PC 로그인 (날짜 계산)
단서: 탁상 달력의 "10월 31일(내 생일)" 마킹 + UV 모드에서 발견되는 "생일로부터 100일째" 메모.

상호작용: 낡은 PC 모니터 클릭 ➔ 4자리 숫자 패드 UI 팝업.

정답 (Input): 0208 (10월 31일 + 100일 = 2월 8일)

클리어 보상: 암호화된 텍스트 문서 열람 권한 해제.

🧩 퍼즐 2: 화학 약품 캐비닛 (카이사르 암호)
단서: 퍼즐 1에서 얻은 PC 문서 (내용: "황제는 3번 찔렸다" = 알파벳 3칸 시프트) + 붉은 글씨 [WRALF].

상호작용: 캐비닛 다이얼 자물쇠 클릭 ➔ 5자리 상하 드래그형 알파벳 다이얼 UI 팝업.

정답 (Input): TOXIC (WRALF를 알파벳 순서대로 3칸씩 앞당김)

클리어 보상: 캐비닛 문이 열리며, '독극물 배합 비율 메모(As:Hg:Tl = 1:2:3)' 획득.

🧩 퍼즐 3: 소형 금고 (마방진 논리)
단서: UV 조명 상태에서 벽에 나타나는 빈칸이 뚫린 3x3 마방진 숫자 + 금고 표면의 힌트 "Missing Numbers + The Sum".

상호작용: 소형 금고 클릭 ➔ 디지털 숫자 패드 UI 팝업.

정답 (Input): 5215 (마방진 빈칸에 들어갈 숫자 5, 2 + 모든 줄의 합계인 15)

클리어 보상: 금고가 열리며 '주기율표 조각' 획득.

🧩 퍼즐 4: 철문 최종 탈출 (정보 조합)
단서: 1. 벽에 걸린 사진 3장 뒷면의 UV 형광 글씨 (As, Hg, Tl).
2. 금고에서 얻은 완성된 주기율표(원자 번호 파악: As=33, Hg=80, Tl=81).
3. 캐비닛에서 얻은 배합 비율 메모 (1:2:3).

상호작용: 철문 도어락 클릭 ➔ 알파벳과 숫자가 혼합된 터치식 풀 키보드 UI 팝업.

정답 (Input): As33Hg160Tl243 (각 원자 번호에 배합 비율 1, 2, 3을 곱한 값)

클리어 보상: "Formula Confirmed" 메시지와 함께 철문이 열리는 사운드 재생 및 게임 클리어(엔딩 씬 전환).
테마 화면

### 기본 상태

**배경**: 방탈출 카페 로비, 중앙 벽면에 네온 사인의 타이틀, 책상 위 방탈출의 상징들(기능 X)
**사운드**: 잔잔한 느낌의 BGM 무한 재생
**폰트**: 고딕체

### 버튼 상호작용

화면 하단에 텍스트 버튼 3개 테마 입장, 환경 설정, 나가기 등
**마우스 호버 시**
- 텍스트 색상이 흰색→네온 시안(#00FFFF)으로 변경, 텍스트 크기 1정도 커짐
**마우스 이탈 시**
- 텍스트의 색상 및 크기가 기본 상태로 복구 됨
**마우스 클릭 시**
- 테마 입장: 씬 전환 연출 실행
- 환경 설정: 설정 팝업 활성화(볼륨 조절 등)
- 나가기: 게임 종료

### 씬 전환 연출(테마 입장 클릭 시)

**사운드**
- 카운터 종소리
- 기본 BGM 빼기(페이드 아웃으로 천천히)
**시각 효과**
- 페이드 아웃으로 까매지면서 테마 선택 화면으로 전환

## 테마 선택 화면

### 기본 상태

**배경**
- 벽면에 정면으로 걸린 포스터들, 잠긴 테마 없음
- 난이도 순으로 정리된 포스터들
**시각 효과**
- 이전 화면에서 넘어올 때 페이드 인으로 화면 전환

### 포스터 상호작용

벽면에 걸린 포스터들에 대한 마우스 이벤트

**마우스 호버 시**
- 밝기 증가 (10%)
- 포스터 전체 크기 1.05배 확대
**마우스 이탈 시**
- 기본 상태로 복구
**마우스 클릭 시**
- 테마 정보 팝업 활성화

### 테마 정보 팝업

**등장 연출**
- 팝업창이 들어오면서 선택한 포스터를 뺀 다른 배경이 살짝 어두워짐
- 각 테마들의 특징에 맞는 BGM 사용(ex)살인마의 암실의 서늘하고 긴장감 있는 BGM)

### 버튼 상호작용

**테마 시작 클릭 시**
- 게임 진입 연출
- 철문 닫히는 효과음
-페이드 아웃으로 테마 방 입장

**뒤로 가기 클릭 시**
- 팝업창 사라짐
- 배경 밝기 복구
## 게임 개요

테마 컨셉: 연쇄 살인마의 암실에 갇힌 형사가 되어, 방 안의 단서를 조합해 탈출하는 테마
승리 조건: 살인마가 돌아오기 전(제한 시간 내) 암실들의 단서를 조합해 최종 락인 철문 비번을 알아내어 입력하는 것
패배 조건: 제한 시간 초과 시 게임 오버
특징: 텍스트를 통한 직접적인 스토리텔링 보단, 유저가 상황을 스스로 추리하게 만드는 방식

## 화면 구성 및 기믹

### 단일 화면 구성 (정면)

**좌측**
- 약품 캐비닛: 굳게 닫혀 있음, 문 손잡이에 5자리 알파벳 다이얼 자물쇠가 걸려 있음
- 벽면 포스터: 낡은 주기율표가 붙어있음 (일부 조각이 찢김)
**중앙**
- 책상: 전원이 켜진 낡은 PC 모니터 (비밀번호 입력), 우측에 탁상 달력
- 벽면: 조명을 바꿀 수 있는 스위치, UV 조명일 경우 빈 벽에 3x3 마방진 생김
**우측**
- 선반: 소형 금고 (클릭 시 4자리 숫자 패드 팝업)
- 벽면: 단서를 조합할 수 있는 보드
- 출구: 잠긴 철문과 알파벳과 숫자가 조합된 잠금 장치

### 핵심 기믹

**조명 전환 시스템**
- 조명 스위치를 클릭할 때마다 방 전체 조명이 변함

**어두운 조명 상태**: 암실의 기본 상태로 일반적인 사물과 텍스트 단서를 볼 수 있다.

**UV 라이트 상태**: 보라색 조명 상태로 살인마가 남긴 단서들이 화면 곳곳에 드러난다.

## 상세 퍼즐

**보이지 않는 흔적 (핵심 기믹 학습)**
- 시작 직후, 힌트가 부족한 상황
- 유저가 방을 탐색하다가 중앙의 조명 스위치를 눌러 조명 변경
- 중앙 벽면에 3x3 마방진 그리드의 일부 숫자, 탁상 달력에는 “그 녀석은 내 생일 기념으로 죽이자”라는 숨겨진 문장이 드러남

**PC 로그인**
- 책상의 낡은 PC 모니터에 4자리 숫자 비번이 있음 비번 힌트로는 “*내 생일*”이라는 문구
- 일반 조명 상태에서 달력 클릭 시 찢어진 달력 발견 → 4월 21일에 D-17
- 4월 21일에서 17일을 더함. **5월 8일,** PC 팝업에 **[0508]** 입력하여 잠금 해제
- PC 화면에서 다음 퍼즐의 단서인 암호화된 텍스트 문서 열람 가능

 **약품 캐비닛**
- 화면 좌측 캐비닛에 5자리 알파벳 다이얼 자물쇠가 걸려 있음
- 열람한 PC 문서 제목 “황제는 세 번의 칼에 찔려 쓰러졌다.(The Emperor fell by 3 stabs.)”
→ 카이사르 암호 암시(알파벳을 3칸씩 미는 암호)
- 문서 본문에 적힌 붉은 글씨 **[W R A L F]**, 카이사르 암호 해석 시 **[T O X I C]**을 입력하여 잠금 해제
- 캐비닛이 열리고 안에서 찢어진 피해자의 사진 3장 획득

**금고**
- 방 안의 소형 금고 4자리 숫자를 입력 / 금고 표면 Missing Numbers + The sum 단서
- UV 조명으로 발견한 벽면의 3x3 숫자 그리드, 밑에 사진 참고 중간 중간 숫자 빠짐
- 가로, 세로, 대각선의 합이 15가 되는 전통적인 마방진 규칙 파악 후 [빠진 숫자1, ‘’2, 1, 5] 입력
- 금고 안에서 주기율 표 조각 획득

![image.png](attachment:3c19452b-9b33-4420-ba87-beee744d0d14:image.png)

**최종 탈출 (주기율 표)**
- 앞에서 얻은 사진 3장 뒤에 피해자마다 사용된 독극물의 원소 기호가 써져 있음 (UV 조명 사용 시)
- 영문자 및 숫자를 조합하는 패드를 풀어야 함.
- 보드에 피해 순서대로 걸려있는 사진을 통해 순서를 특정하고, 사진 뒷면의 증상과 연관된 독극물을 찾아 비밀번호를 푼다. (이때 메모에 사용한 독뿐만이 아닌 여러 개의 독극물에 대한 힌트들이 있음)
- 사진 1 뒤엔 급성 중독 증상이 보임, 사진 2 뒤 신경 이상 증상이 보임, 사진  3 뒤엔 호흡에 이상이 있어 보임 등 독극물을 예측 후 주기율 표에서 번호 확인 (순서대로 비소, 수은, 염소)
- 키패드에 **[As 33 Hg 80 Cl 17]** 입력
